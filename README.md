# dope-theme
iOS icon theme (tested with iOS 9) compatible with Anemone.

Currently has 18 icons: 
  - Alien Blue
  - Allo
  - App Store
  - Appster
  - iBooks 
  - Camera
  - Calendar
  - Facebook Messenger
  - Google Hangouts
  - Google Home
  - Google Maps
  - Messages
  - Music
  - Phone
  - Safari
  - Settings
  - Snapchat
  - YouTube
